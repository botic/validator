Module validator/stick-validator

Middleware to validate requests. Depending on the request method it automatically decides if it uses req.queryParams (default) or req.postParams (only for POST and PUT). The validator requires the params middleware to be configured.

Example 

app.configure("params", require("stick-validator"), "route");

app.get("/", function(req) {
  req.validate("countryCode", true).hasLength(3, "Invalid country code!");
  req.validate("email").isDefined("Email missing!").isEmail("Invalid email!");

  if (req.hasFailures()) {
    return response.html(req.failureMessages().join("<br>"));
  }

  return response.html("Ok");
});

Returns the validation failure messages.

Parameters

String name Optional, name to limit the messages to a single parameter

Returns

Array An array containing all failure messages

request.hasFailures (name)

Returns true if any of the validations of this validator failed.

Parameters

String name Optional, name of a single parameter to check

Returns

Boolean True if any (or the specified parameter) validation failed

request.validate (name, trim)

The returned Validation instance contains various validation functions that can be chained. Note that this validation object stops validating at the first fail.

Parameters

String name The name of the request parameter
Boolean trim Optional, trim the value of the parameter

Returns

Validation A Validation instance
