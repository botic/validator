Module validator

A module to validate and sanitize an object’s properties in RingoJS. It depends on Ringo’s ringo/utils/strings module for string validation.

Example 

var validator = new Validator({
  "email": "foo.bar@example.com"
});

// validate a value
validator.validate("email").
         isDefined("Email address is missing!").
         isEmail("Invalid email address provided");

// returns true if a validation failed
validator.hasFailures();

// returns an array with all error messages
validator.getMessages();
