Validator (objectToValidate)

Creates a new Validator instance

Example 

var validator = new Validator({
  "email": "foo.bar@example.com"
});

// validate a value
validator.validate("email").
         isDefined("Email address is missing!").
         isEmail("Invalid email address provided");

// returns true if a validation failed
validator.hasErrors();

// returns an array with all error messages
validator.errorMessages();

// returns an object suitable for use as rendering context
validator.getTemplateContext();

Parameters

objectToValidate

Returns

Validator A newly created validator instance

Validator.prototype.errorMessages (key)

Returns the validation failure messages.

Parameters

String key Optional key to limit the failure messages to a single validation

Returns

Array An array containing all failure messages

Validator.prototype.getTemplateContext ()

Returns an object suitable for use as template rendering context. This object contains all values stored by their name, an object named errors containing the validation failure messages by value name, and a boolean hasErrors.

Returns

Object The template context object
