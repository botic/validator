Modules: Validator API

  • validator

    A module to validate and sanitize an object’s properties in RingoJS. It depends on Ringo’s ringo/utils/strings module for string validation.

  • validator/stick-validator

    Middleware to validate requests. Depending on the request method it automatically decides if it uses req.queryParams (default) or req.postParams (only for POST and PUT). The validator requires the params middleware to be configured.
